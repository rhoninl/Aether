# Aether

Aether is a Rust-based engine workspace with multiple crates for input handling, world runtime, and rendering.

## Prerequisites
- Rust toolchain (stable)
- Cargo (comes with Rust)

## Build
```bash
cargo build
```

## Test
```bash
cargo test
```

## Workspace
The repository uses a multi-crate Cargo workspace. Main workspace crates are under `crates/`.

## Notes
- Run commands from the repository root.
- This branch currently reflects ongoing development work from a feature branch (`claude-001/ecs-core`).
